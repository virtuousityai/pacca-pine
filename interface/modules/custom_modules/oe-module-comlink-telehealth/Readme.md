### Comlink Telehealth Module

This project implements the Comlink Telehealth system for video conferencing for Pacca PINE installations.

The instructions for using this module can be found on the Pacca PINE Wiki page here [Comlink Telehealth Technical Guide](https://www.open-emr.org/wiki/index.php/Comlink_Telehealth)
